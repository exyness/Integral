# Product Overview

Integral is a comprehensive personal productivity suite that unifies task management, note-taking, time tracking, journaling, and budgeting in a single platform.

## Core Features

- **Dashboard**: Central hub with unified overview of productivity metrics and quick actions
- **Tasks**: Project-based task management with priorities, due dates, tags, Kanban boards, and productivity insights
- **Notes**: Rich text editor (Lexical) with folder organization, pinning, favorites, and instant search
- **Time Tracking**: Activity logging with detailed session history and persistent floating timer widget
- **Pomodoro**: Customizable focus timer with work/break intervals, session history, and notes
- **Journal**: Daily reflections with calendar navigation, mood tracking, and activity tags
- **Budget**: Multi-currency financial tracking with analytics, budgets, and transaction management
- **Accounts**: Secure credential storage with usage tracking and organized folders

## Halloween Edition

The application features a special Halloween theme mode with:

- Animated decorations (floating ghosts, flying witches, hanging spiders, glowing pumpkins)
- Teal color scheme (#60c9b6) with Creepster font
- Themed empty states for every page
- Background music and visual effects
- Toggle between normal and Halloween modes

## Target Users

Productivity enthusiasts who want an all-in-one solution for personal organization, time management, and financial tracking with a modern, visually appealing interface.
